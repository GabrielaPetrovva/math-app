# Комбинаторика — Калкулатор & Учебник

Интерактивно уеб приложение за изучаване и изчисляване на пермутации, комбинации и вариации.

## Структура на проекта

```
├── index.html   # Структура и съдържание
├── style.css    # Стилове (DM Sans, DM Serif Display, DM Mono)
└── app.js       # Логика — калкулатор, учебник, задачи
```

## Функционалности

### Калкулатор
- Изчислява **пермутации** P(n) = n!, **комбинации** C(n,k) = n! / (k! · (n−k)!) и **вариации** V(n,k) = n! / (n−k)!
- Показва стъпка по стъпка как е изчислен резултатът
- Пази история на последните 8 изчисления
- Валидира входните стойности (n от 0 до 20, k ≤ n)

### Учебник
- Теория с обяснения и реални примери за трите вида комбинаторни обекти
- Визуално дърво за пермутации
- Сравнителна таблица между комбинации и вариации
- Формули за връзката между трите понятия

### Задачи
- 39 задачи разпределени по тема (P / C / V) и ниво (лесно / средно / трудно)
- Проверка на отговора с обратна връзка
- Филтриране по тема и ниво
- Прогресът се запазва в `localStorage`
- Опция за нулиране на прогреса

## Технологии

- Чист HTML / CSS / JavaScript — без фреймуъри или зависимости

## Как да стартираш

Отвори `index.html` в браузър — не е нужен сървър.

```bash
# Или с локален сървър (по избор)
npx serve .
```

## Поддържани браузъри

Всички модерни браузъри (Chrome, Firefox, Safari, Edge). Адаптивен дизайн за мобилни устройства.
